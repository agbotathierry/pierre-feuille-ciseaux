# Pierre-Feuille-Ciseaux

Un jeu de Pierre-Feuille-Ciseaux en JavaScript joué dans la console du navigateur.

## Comment jouer

1. Ouvre `index.html` dans ton navigateur
2. Ouvre la console (F12)(plus besoin de console je viens de faire l'interface)
3. Tape ton choix : pierre, feuille ou ciseaux(je viens de faire les bouton alors)
4. Le jeu se joue en 5 manches
5. Le vainqueur est annoncé à la fin !

## Technologies utilisées

- HTML
- JavaScript
- css
